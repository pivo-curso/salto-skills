# Salto para o Dólar — Skills

Ferramentas prontas de **Claude Code** do curso **Salto para o Dólar / Pivô**. Instala uma vez e usa direto no seu Claude.

## O que vem
- **`/portfolio`** — monta seu portfólio no padrão gringo (que fecha em dólar): em inglês nativo, posicionado por resultado, com o método da Clarisse (as 7 coisas inegociáveis).
- **`/proposta`** — escreve uma proposta comercial vencedora em inglês pra vaga/cliente gringo (pesquisa o cliente → SPIN → anatomia 30/20/30/20 → preço ancorado no resultado).

## Como instalar
No Claude Code, rode os dois comandos:

```
/plugin marketplace add SEU-USUARIO/salto-skills
/plugin install salto@salto-skills
```

Pronto. Agora é só usar:

```
/portfolio
/proposta
```

## Atualizar (quando sair melhoria)
```
/plugin marketplace update salto-skills
```
